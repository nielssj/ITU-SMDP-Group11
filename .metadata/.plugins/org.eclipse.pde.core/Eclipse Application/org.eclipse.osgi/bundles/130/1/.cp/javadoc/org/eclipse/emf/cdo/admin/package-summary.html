Package org.eclipse.emf.cdo.admin

Client side of the protocol to administer CDO repositories remotely.


Interface Summary
CDOAdminClient A client-side administrative interface.
CDOAdminClient.ConnectionStateChangedEvent An event fired from a client-side administrative interface after the connection state has changed.
CDOAdminClientManager Manages multiple remote administration connections.
CDOAdminClientRepository A client-side administrative interface to a remote repository.
CDOAdminClientRepository.SessionConfigurator Provides access to the session configuration used in calls to CDOAdminClientRepository.openSession().
 

Class Summary
CDOAdminClientUtil Various static methods that may help with CDO remote administration.
