Package org.eclipse.emf.cdo.transfer.ui

Reusable implementatins of common JFace concepts for dealing with transfers of resources between systems.


Class Summary
TransferContentProvider A reusable tree content provider for the mappings of a transfer.
TransferDialog A dialog that contains a TransferComposite.
TransferDragListener A drag listener that can support a viewer to create and perform transfers of the dragged elements to target elements.
TransferDropAdapter A drop adapter that can support a viewer to create and perform transfers of the dragged elements to target elements.
TransferLabelProvider A reusable table label provider for the mappings of a transfer.
TransferTypeContentProvider A reusable content provider for the transfer types of a mapping.
